# VIT Pune AI Agent - Capstone Project Group 7

> Generative AI Agent for University Student Support
> Built with RAG + LangChain + LLaMA3 + ChromaDB + Streamlit

---

## Project Structure

```text
university_ai_agent/
|-- app.py
|-- rag_engine.py
|-- requirements.txt
|-- README.md
|-- .gitignore
|-- data/
|   |-- university_data.txt
|   |-- feedback.csv
|   `-- deadlines.txt
`-- vectorstore/
```

Notes:
- `data/` stores the source files used by the app.
- `vectorstore/` is auto-generated by ChromaDB when the agent is initialized.
- `__pycache__/` and similar cache folders should not be kept in the project structure.

## Setup

### 1. Install Python 3.10+
Download from: https://www.python.org/downloads/

### 2. Install Ollama
Download from: https://ollama.ai

### 3. Pull the LLaMA3 model

```bash
ollama pull llama3
```

### 4. Install dependencies

```bash
pip install -r requirements.txt
```

### 5. Run the app

```bash
streamlit run app.py
```

The app opens at `http://localhost:8501`.

### 6. Initialize the agent
Use the sidebar button inside the app. The first run builds `vectorstore/`; later runs reuse it.

---

## Troubleshooting

- If Ollama is not running, start it with `ollama serve`.
- If the model is missing, run `ollama pull llama3`.
- If port `8501` is busy, run `streamlit run app.py --server.port 8502`.
